# 🛫 Trip Nest - Backend Server

This is the backend server for **Trip Nest**, a full-stack travel booking platform. The backend is built using **Node.js**, **Express**, and **MongoDB**, providing RESTful APIs for managing tour packages, bookings, user data, and authentication.

---

## 📦 Tech Stack

- **Node.js** ^18+
- **Express** ^5.1.0
- **MongoDB** ^6.16.0 (using MongoDB Native Driver)
- **CORS** ^2.8.5
- **dotenv** ^16.5.0

---

## 🚀 Features

- 🌍 RESTful API Endpoints
- 🔐 Secured with Environment Variables
- ⚙️ Express Middleware for Request Handling
- 📡 CORS Enabled for Frontend Communication
- 📁 MongoDB Database Connection
